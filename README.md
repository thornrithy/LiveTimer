# ⏰ Live Clock

A modern and customizable productivity web application designed to help users stay focused and organized. Live Clock combines a real-time digital clock with productivity tools such as a countdown timer, Pomodoro timer, task management, and a built-in YouTube music player—all within a clean and responsive interface.

---

## 🎯 Project Goal

The goal of Live Clock is to provide an all-in-one productivity workspace where users can manage their time, organize daily tasks, and create a personalized working environment that improves focus and efficiency.

---

## ✨ Features

### 🕒 Time Management
- Real-time digital clock
- Live date display
- 12/24-hour time format
- Countdown timer
- Custom countdown timer
- Pomodoro timer
- Start, pause, and reset timer controls

### ✅ Task Management
- Create and delete tasks
- Multiple task icons
- Task counter
- Interactive to-do list

### 🎵 Music Player
- Built-in YouTube player
- Add YouTube videos using URLs
- Playlist management

### 🎨 Customization
- Multiple gradient backgrounds
- Image backgrounds
- GIF backgrounds
- Upload custom images or GIFs
- Custom background URL
- Multiple clock fonts
- Adjustable font size
- Personalized interface

### 📱 User Experience
- Responsive design
- Modern UI
- Sidebar navigation
- Bootstrap 5 layout
- Font Awesome icons

## 🚀 Getting Started

1. Clone the repository.

```bash
git clone https://github.com/your-username/live-clock.git
```

2. Open the project folder.

3. Launch `index.html` using Live Server or any local web server.

4. Start managing your time and tasks.
